# 图书管理系统简介 java1059

## 项目概述

本系统是一个基于SpringBoot和Vue3的图书管理系统，采用前后端分离的设计模式。系统主要包括管理员和用户两个角色，提供了图书信息管理、借阅管理、用户管理等丰富的功能模块。

## 技术栈

- 后端：Spring Boot
- 前端：Vue3
- 数据库：MySQL
- 构建工具：Maven
- 开发环境：IDEA、Node.js

## 功能模块

### 系统角色

- 管理员
  - 图书信息管理
  - 图书类型管理
  - 借阅信息管理
  - 用户管理
  - 密码更改
- 用户
  - 图书管理
  - 借阅图书管理
  - 其他管理（修改密码）

## 运行环境

- JDK 1.8
- MySQL 5.5
- IDEA
- Node.js
- Maven

## 目录结构

```
项目目录
│
├── 后端（Spring Boot）
│   ├── src
│   ├── pom.xml
│   └── ...
└── 前端（Vue3）
    ├── src
    ├── package.json
    └── ...
```

## 核心亮点

- 前后端分离，便于维护和扩展
- 使用Spring Boot和Vue3，确保项目的高效与稳定性
- 集成Mybatis，简化数据库操作
- 采用ElementUI，界面友好，操作简便

## 部署步骤

1. 准备运行环境：JDK 1.8、MySQL 5.5、IDEA、Node.js、Maven
2. 克隆项目源码
3. 导入SQL文件，创建数据库和表
4. 分别编译后端和前端代码
5. 部署后端服务
6. 部署前端服务
